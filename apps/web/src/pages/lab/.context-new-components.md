# Контекст: Новые компоненты (New Components Lab)

## Флоу работы

### Структура
- `pages/lab/new-components.astro` — страница-шоурум для новых компонентов
- `components/lab-new/` — хранилище новых компонентов (НЕ отдельный UI-kit)
- `components/ui/` — текущие компоненты, НЕ ТРОГАТЬ

### Процесс
1. Создаём компонент в `components/lab-new/ComponentName.astro`
2. Импортируем и добавляем на `lab/new-components.astro`
3. Утверждаем, правим
4. Когда готово — мигрируем в `components/ui/`

### Доступ
- `http://localhost:4321/lab/new-components`

### 1. Glassmorphism (страница /lab/glass)
- Использовать `backdrop-blur-md bg-white/15 border border-white/20`
- Базовый вариант: `backdrop-blur-md bg-white/15 border border-white/20 rounded-2xl shadow-xl`

### 2. Карточка с изображением на весь размер
**Выбран вариант 1:** фоновое изображение + градиент + абсолютный текст

```html
<div class="relative h-96 rounded-2xl overflow-hidden group">
  <!-- Изображение на весь размер -->
  <img src="..." class="absolute inset-0 -z-10 w-full h-full object-cover" />

  <!-- Градиент для читаемости текста -->
  <div class="absolute inset-0 bg-gradient-to-t from-black/60 via-black/20 to-transparent" />

  <!-- Текст поверх -->
  <div class="relative h-full p-8 flex flex-col justify-end">
    <h3 class="text-white text-2xl">Заголовок</h3>
    <p class="text-white/80 mt-2">Описание</p>
  </div>
</div>
```

## Правила и ограничения

### Tailwind — можно почти всё
- **Arbitrary values:** `w-[137px]`, `rounded-[12px]`, `shadow-[0_4px_24px_rgba(0,0,0,0.12)]`
- **Произвольные свойства:** `[backdrop-filter:blur(20px)]`
- **Расширение через tailwind.config:** цвета, шрифты, spacing, borderRadius, boxShadow
- **global.css через @layer:** для своих классов

### НЕ делать
- Новые UI-библиотеки (React/Vue/Svelte компоненты)
- CSS-in-JS, Styled Components
- Анимационные библиотеки (GSAP, Framer Motion и т.д.)
- Отдельные .css файлы для каждого компонента

### Предупреждать если
- Сложные SVG-фильтры / SVG-маски
- 3D-трансформации с перспективой
- Спец. анимации
- Любые JS-библиотеки
